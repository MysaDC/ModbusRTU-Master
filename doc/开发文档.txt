本项目旨在开发一个基于 Java 的轻量级数据采集服务（Data Acquisition Service），用于通过 Modbus TCP 协议读取 QB6000 气体报警控制器 的实时数据，并将其转换为 MQTT 消息发布到mqtt broker中。
核心功能:
周期性轮询：通过 Modbus TCP 读取控制器的寄存器数据。
协议转换：将 Modbus 寄存器值解析为物理量（本项目均为气体探头,取得值为气体浓度）。
数据上报：将解析后的 JSON 数据发布到指定的 MQTT Topic。
异常处理：处理网络中断、超时、校验错误等异常情况。
日志输出：数据读取到上报全流程追踪。
项目整体架构见硬件架构图,本项目作为串口服务器到应急管理局监控平台里包含的mqtt broker中间件的中间件提供纯后端服务。通信协议见"QB6000控制器通信协议.txt"。
